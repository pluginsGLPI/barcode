1 - Presentation

This plugin allow you to generate automatically a bars code with avery format.
Compatibility 0.70

2 - Installation

Really easy to do :
- Download the tarball of this plugin
- Uncompress it in the plugins directory of your glpi installation
- To put the logo of your company in the folder /pics/ under the name of logo.png 
(it is necessary that the file is well cavity with format png if not that does not work)
to have the logo of your company during the generation of the codes bars

And the turn and played
